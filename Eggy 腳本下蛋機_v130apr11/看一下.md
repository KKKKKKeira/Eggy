# 🥚 EGGY 腳本下蛋機

> 如果你的便便也像下蛋一樣，那你要多喝水。

快速產出提案用的 AI 分鏡圖，單一 HTML 檔案，不需安裝任何東西。

---

## 使用方法

### 第一次使用

1. **取得 Gemini API Key**
   前往 [https://aistudio.google.com/apikey](https://aistudio.google.com/apikey) 免費申請

2. **雙擊 `storyboard-generator.html` 開啟工具**
   （建議用 Chrome 或 Edge）

3. 首次進入會看到設定畫面，貼上你的 API Key 點「開始使用」
   Key 會存在瀏覽器裡，下次開啟不用重填

### 開始生成

4. 選擇視覺風格、填寫角色設定與故事大綱
5. 逐格填寫分鏡描述
6. 點「🚀 開始生成」，圖片會一張張跑出來
7. 完成後點「📦 全部下載」匯出 ZIP

---

## 功能

- **12 種視覺風格** — 廣告基本款、日式漫畫、皮克斯 3D、JOJO 風、教父風、Kill Bill 等
- **角色參考照** — 上傳角色照片讓 AI 對照臉部外觀
- **場景參考照** — 全域場景底圖 + 每格個別場景圖
- **局部重新生成** — 框選畫面某個區域單獨重生
- **補充描述修版** — 對已生成圖片追加指示重跑
- **ZIP 打包下載** — 圖片 + Prompt 紀錄一起打包

---

## 注意事項

- 需要網路連線（載入介面元件 + 呼叫 Gemini API）
- Safari 用戶在 `file://` 協定下 localStorage 可能受限，建議用 Chrome / Edge
- API Key 只存在你的瀏覽器，不會上傳到任何伺服器
- Gemini 圖片生成 API 超過免費額度後會計費，費用不包含在 Google 訂閱中
- 4K 解析度 / Gemini 3.1 模型會消耗較多 API token

## 技術
